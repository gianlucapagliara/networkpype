Welcome to NetworkPype's documentation!
====================================

NetworkPype is a Python library for handling network operations with a pipeline-based approach.

Features
--------

* REST API request handling with rate limiting
* WebSocket connection management
* Time synchronization capabilities
* Modular pipeline architecture
* Type-safe implementations
* Comprehensive test coverage

Installation
-----------

To install NetworkPype, run this command in your terminal:

.. code-block:: console

   $ poetry add networkpype

For development installation:

.. code-block:: console

   $ git clone https://github.com/yourusername/networkpype.git
   $ cd networkpype
   $ poetry install
